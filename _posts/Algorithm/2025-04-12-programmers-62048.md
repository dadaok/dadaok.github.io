유클리드 호제법(최대공약수)로 푼다.  
실제 도형을 그려보고 겹치는 부분을 계산해 공식을 유추한다.  
주요 공식 : 선이 지나는 가로 칸 + 선이 지나는 세로 칸 - 중첩으로 지나는 칸(세로칸 or 가로칸)

```java
class Solution {
    public long solution(int w, int h) {
        long width = (long) w;
        long height = (long) h;

        return width*height - (width+height-gcd(width,height));
    }

    private long gcd(long w, long h){
        while(h != 0){
            long temp = w % h;
            w = h;
            h = temp;
        }
        return w;
    }
}
```
